# 🚀 Швидкий старт проєкту

## ⚡ Найпростіший спосіб (з SQLite, без PostgreSQL)

### Крок 1: Встановлення Backend залежностей

```bash
cd backend
pip install -r requirements.txt
```

### Крок 2: Ініціалізація бази даних

```bash
# Створити користувачів
python init_db_simple.py

# Завантажити 100 реальних фільмів з TMDb
python import_movies_tmdb.py
```

### Крок 3: Запуск Backend

```bash
uvicorn app.main:app --reload
```

Backend: http://localhost:8000

---

### Крок 4: Встановлення Frontend залежностей

**В НОВОМУ терміналі:**

```bash
cd frontend
npm install
```

### Крок 5: Запуск Frontend

```bash
npm run dev
```

Frontend: http://localhost:3000

## 📦 Що робить кожен скрипт?

### `init_db_simple.py`
- Створює SQLite базу даних
- Додає 3 тестових користувачів
- Швидко і без PostgreSQL

### `import_movies_tmdb.py`
- Завантажує 100 популярних фільмів з TMDb API
- Додає постери, описи, жанри
- Аналізує емоційні характеристики
- Займає ~2-3 хвилини

---

## 🔧 Налаштування кількості фільмів

Відредагуйте `import_movies_tmdb.py`:

```python
# Змініть num_pages (1 сторінка = 20 фільмів)
import_movies(num_pages=5)  # 100 фільмів
import_movies(num_pages=10) # 200 фільмів
```

## ❓ Проблеми?

### Backend не запускається
```bash
# Перевірте чи встановлені всі залежності
pip install -r requirements.txt
```

### Немає фільмів
```bash
# Запустіть імпорт
python import_movies_tmdb.py
```

### Frontend не запускається
```bash
# Видаліть node_modules і перевстановіть
rm -rf node_modules package-lock.json
npm install
```
